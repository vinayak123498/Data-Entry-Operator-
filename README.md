# 📊 Sales Data Management & MIS Reporting Dashboard

## 📌 Overview

This project simulates a real-world MIS Executive role by managing and analyzing sales data using Microsoft Excel.

## 🎯 Objective

To create a structured reporting system for tracking and analyzing sales performance across categories, cities, and salespersons.

## 📂 Dataset

* 50+ sales records including product, category, quantity, price, and location.

## ⚙️ Features

* Data entry and data cleaning
* Excel formulas (SUM, IF, VLOOKUP)
* Pivot tables for analysis
* Dashboard with KPIs and charts

## 📊 Dashboard Includes

* Total Sales
* Total Orders
* Sales by Category (Bar Chart)
* Sales by City (Pie Chart)

## 🛠️ Tools Used

* Microsoft Excel

## 📈 Key Skills Demonstrated

* Data Cleaning
* Data Analysis
* MIS Reporting
* Dashboard Creation
* Excel Automation

## 🎯 Outcome

Developed a complete MIS reporting system that helps in business decision-making using data visualization.
