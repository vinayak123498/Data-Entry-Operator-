# 📊 Sales Data Management & MIS Reporting Dashboard

## 📌 Overview

This project simulates a real-world MIS Executive role by managing and analyzing sales data using Microsoft Excel.

## 🎯 Objective

To create a structured reporting system for tracking and analyzing sales performance across categories, cities, and salespersons.

## 📂 Dataset

* 50+ sales records including product, category, quantity, price, and location.

## ⚙️ Features

* Data entry and data cleaning
* Excel formulas (SUM, IF, VLOOKUP)
* Pivot tables for analysis
* Dashboard with KPIs and charts

## 📊 Dashboard Includes

* Total Sales
* Total Orders
* Sales by Category (Bar Chart)
* Sales by City (Pie Chart)

## 🛠️ Tools Used

* Microsoft Excel

## 📈 Key Skills Demonstrated

* Data Cleaning
* Data Analysis
* MIS Reporting
* Dashboard Creation
* Excel Automation

## 🎯 Outcome

Developed a complete MIS reporting system that helps in business decision-making using data visualization.

# 💼 Employee Salary MIS System (Excel)

## 📌 Overview

This project simulates an HR/Accounts MIS system used to calculate employee salaries and bonuses using Excel functions.

## 🎯 Objective

To automate salary and bonus calculation using lookup functions and generate reports for management.

## 📂 Dataset

* 50 employee records with department and working hours.

## ⚙️ Features

* Salary calculation using VLOOKUP / INDEX-MATCH
* Bonus calculation using lookup functions
* Data automation and validation
* Pivot tables for department-wise analysis

## 📊 Dashboard Includes

* Total Salary Expense
* Total Bonus Paid
* Salary by Department (Bar Chart)
* Bonus Distribution (Pie Chart)

## 🛠️ Tools Used

* Microsoft Excel

## 📈 Key Skills Demonstrated

* VLOOKUP
* INDEX & MATCH (XLOOKUP alternative)
* Data Processing
* MIS Reporting
* Dashboard Design

## 🎯 Outcome

Built an automated payroll MIS system that reduces manual work and improves accuracy in salary processing.

# 📦 Inventory & Stock Management MIS Dashboard

## 📌 Overview

This project simulates a real-time inventory management system used in retail and warehouse operations.

## 🎯 Objective

To track stock movement, identify low stock items, and generate reports for efficient inventory management.

## 📂 Dataset

* 50 stock transaction records including product, quantity in/out, and dates.

## ⚙️ Features

* Stock calculation (In Qty – Out Qty)
* Product lookup using VLOOKUP / INDEX-MATCH
* Stock value calculation
* Low stock alert using IF function
* Pivot tables for analysis
* Interactive dashboard with slicers

## 📊 Dashboard Includes

* Total Stock Value
* Total Products
* Low Stock Count
* Stock by Category (Bar Chart)
* Stock Status (Pie Chart)
* Slicer filters for dynamic analysis

## 🛠️ Tools Used

* Microsoft Excel

## 📈 Key Skills Demonstrated

* Inventory Management
* Data Analysis
* Excel Functions (VLOOKUP, IF)
* Dashboard Creation
* Reporting & Visualization

## 🎯 Outcome

Created a dynamic inventory MIS system that helps monitor stock levels and supports decision-making.
